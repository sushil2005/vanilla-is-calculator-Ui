# 🧮 Vanilla JS Calculator UI

A modern and responsive calculator built using **HTML, CSS, and Vanilla JavaScript**.  
This project is designed to practice DOM manipulation, event handling, and UI design fundamentals.

---

## 🚀 Features

- Clean and modern UI design
- Basic arithmetic operations (+, -, ×, ÷)
- Responsive layout
- Error handling (division by zero)
- Real-time result display

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6)
